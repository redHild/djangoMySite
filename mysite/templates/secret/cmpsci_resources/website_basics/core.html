Curriculum for Website Basics
Jakob C. Hild & Marguerite Weldon 2020
Unit 1
HTML
  1. Introduce Basic HTML structure and create "Hello World" Program.
  2. Discuss various HTML Elements.
  3. Discuss additional HTML structures, such as lists and tables.
  4. Add functionality to jump from one HTML page to another.
  5. Discuss Forms and buttons and text areas.
Unit 2
CSS
  1. Discuss the various ways of implementing style sheets using HTML.
  2. Introduce CSS Parameters and values.
  3. Give time for student to play with CSS Styling.
Unit 3
JavaScript
  1. Introduce JS variables and connecting it to HTML.
  2. Work into more and more complex Coding exercises, which add functionality using various Programming Ideas (Functions, Classes, Variable Manipulation, etc...).
Unit 4
PHP
  1. Discuss PHP syntax and various use cases.
  2. Discuss Server commands... POST and GET.
  3. XML and JSON file parsing.
