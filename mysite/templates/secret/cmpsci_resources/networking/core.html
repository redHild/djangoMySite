Curriculum for Networking
Jakob C. Hild & Marguerite Weldon 2020
Unit 1
Intro & DNS
  1. [Intro] Introduce the OSI model with it's various layers. With a brief introduction to Protocols and the history of the internet.
  2. [DNS] Describe the Domain Name System's history and how it works. Topics include sub-domains, top-level domains (TLD), how to do a DNS lookup, and the concept of DNS Caching.
  3. [Homework 1] The topics covered in this unit will be enough to complete this set of exercises.
Unit 2
HTTP
  1. [HTTP] Go into the detail of HyperText Transfer Protocol and it's syntax. Describe it's commands and headers and resulting status codes. Describe URL's and URL encoding. Learn about GET and POST. Describe Cookies and more about Caching.
  2. [Homework 2] The topics covered in this unit will be enough to complete this set of exercises.
Unit 3
Application Layer
  1. [AppLayer] This will talk about several of the protocols associated with the application layer: TLS, FTP, SSH, Email, and Base64.
  2. [Homework 3] This homework will also test you on the HTTP Notes a little bit more.
Unit 4
Transport Layer
  1. [AppLayer] You should read more into Email, Conversions, and Base64 This time around.
  2. [TrnLayer] This covers Ports, Checksums, and UDP.
  3. [Homework 4] This unit has everything you need.
Unit 5
Transmission Control Protocol
  1. [TCP] This goes into a deep dive of TCP: 3-way-handshake, Closing a connection, reliability, Headers, Flow Control, and more.
  2. [Homework 5] This unit has everything you need.
  3. [Homework 6] There's a lot to talk about in TCP so there are two homeworks for it.
  4. [Homework 7] Let's make that 3.
Test Yourself
Exam 1
  1. [Exam 1] This will use material from all of the notes used up to this point.
Unit 6
Internet Protocol
  1. [IP] This is all about internet protocol and the different types of headers it holds. It also covers fragmentation, ping, trace root and much, much more.
  2. [IPv6] This covers IPv6 and ICMP aka Internet Control Message Protocol.
  3. [IPAddr] This covers how IP addresses were given out in blocks and how networks rely on the information that comes with an IP network.
  4. [Homework 8] This unit has everything you need.
Unit 7
Routing
  1. [NAT/DHCP/APIPA/ARP] We are crawling slowly down the OSI model, now we can finally look at how routers communicate.
  2. [Routing] Routing goes somewhat into graph theory and how routers actually routers can choose the best route from one computer to the next. This also covers a little bit of the ASes or Autonomous Systems and BGP or Border Gateway Protocol that makes up the structure of the Internet.
  3. [Homework 9] This unit has everything you need.
Test Yourself
Exam 2
  1. [Exam 2] This will use material from all of the notes used up to this point.
Unit 8
Physical Link
  1. [PhysLink] At the lowest part of the OSI model we start to look at the physical connections whether it be the frequencies data travels or the actual wires connecting devices. This also get's into how two signals can block each other and the ways we avoid that with collision detection. Also covered is the physical structure of the internet and how the internet is mostly made up of packet switching stations called routers.
  2. [Homework 10] This unit has everything you need.
