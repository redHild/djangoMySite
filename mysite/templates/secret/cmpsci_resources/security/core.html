Curriculum for Security
Jakob C. Hild & Marguerite Weldon 2020
Unit 1
Cryptography Intro
  1. [Classical Cryptography 1] This includes the differing types of cyphers: Substitution, Vigenere, One-Time pad, and how to use or find their keys.
  2. [Homework 1] The topics covered in this unit will be enough to complete this set of exercises.
Unit 2
Permutation and Stream Cyphers
  1. [Classical Cryptography 2] This goes into Permutation Cyphers and the Playfair cypher.
  2. [Modern Cryptography 1] This starts talking about Stream cyphers by using xor against a key and plain text or encrypted text to get its counterpart.
  3. [Homework 2] The topics covered in this unit will be enough to complete this set of exercises.
Unit 3
Block Cyphers and Special Topics
  1. [Modern Cryptography 2] This covers how we measure security, Pseudo Random numbers, and a special topic called the Birthday Problem.
  2. [Modern Cryptography 3] This covers a good amount of block cyphers. These are very important because there is rarely any encryption done without using the special modes of block cyphers. It covers how large blocks should be and The Data Encryption Standard DES Mode and why we never use it and we use Advanced Encryption Standard AES commonly today.
  3. [Homework 3] This homework will also test you on the HTTP Notes a little bit more.
Unit 4
Key Exchange and Hashing
  1. [Modern Cryptography 4] This Covers the various types of key exchange algorithms and attacks to find keys: Diffie-Hellman, RSA, Trapdoor functions, and side channel attacks.
  2. [Modern Cryptography 5] This goes over what a Hash is and how it is used in security. This also covers common hash functions, collisions between hashes, Digital signatures, Certificates, and TLS or Transport Layer Security.
  3. [Homework 4] This unit has everything you need.
  4. [Homework 5] This unit has everything you need.
Test Yourself
Exam 1
  1. [Exam 1] This will use material from all of the notes used up to this point.
Unit 5
Passwords
  1. [Passwords] This is all about password Security and how to crack passwords using hashes. We also learn about how we can strengthen our hashes from brute force algorithms like rainbow table searching using salting. There's also a few NIST password recommendations.
  2. [Homework 6] This unit has everything you need.
Unit 6
DNS Security
  1. [DNS Security] This has to do with much of the security around domain names: Phishing Sites, hosts.txt, Cache Poisoning, Evil Twin Access Points, DoS attacks, and plenty of solutions to each problem.
  2. [Homework 7] This unit has everything you need.
Unit 7
Network Security
  1. [Network Security 1] We cover networking very briefly and discuss, Attacks using TCP, Port Scanning, and how to protect servers from those attacks and scanning.
  2. [Network Security 2] This goes through many many types of attacks. Ping flood, Ping of death, Smurf Attack, ICMP tunneling, Ping sweep, Trace route, ICMP redirect, Disabling ICMP. We also have many types of DoS attacks: DNS amplification, NTP amplification, UDP flood, and the Echo-chargen attack.
  3. [Network Security 3] We learn about: Network Address Translation NAT, Virtual Private Networks VPN, Firewalls for filtering, Wi-Fi security/encryption with WEP, WPA, WPA2, and WPS.
  4. [Homework 8] This unit has everything you need.
  5. [Homework 9] This unit has everything you need.
  6. [Homework 10] This unit has everything you need.

Unit 8
Malware & Buffer Overflow
  1. [Malware & Buffer Overflow] The many types of Malware: Viruses, Macro viruses, Worms, Rootkits, Trojans, Logic Bombs, Backdoors, Spyware, Ransomware, and more. We also get an insight into a type of attack that takes advantage of the buffer overflow error that is sometimes not checked for in things like js or c.
  2. [Homework 11] This unit has everything you need.
